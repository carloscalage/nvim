# Neovim Config

Configuracao pessoal do [LazyVim](https://github.com/LazyVim/LazyVim).

## Setup

Em uma maquina nova, rode:

```bash
git clone https://github.com/carloscalage/nvim.git /tmp/nvim-setup && bash /tmp/nvim-setup/setup.sh
```

O script instala o Neovim, configura o PATH (`~/.zshrc` ou `~/.bashrc`) e sincroniza os plugins automaticamente.

Depois, reinicie o shell e abra o `nvim`.
